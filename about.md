# About GigaElk

GigaElk is a one-person indie software studio run from Tokyo, Japan.

I make things. Games, tools, apps, plugins — whatever catches my interest. Sometimes it's a Unity game for a game jam. Sometimes it's an After Effects plugin for animators. Sometimes it's an AI-powered phone answering system. There's no theme except "that sounds fun, let me try it."

## What I Do

I'm a full-stack developer with 20+ years of experience building software across a wild range of domains:

- **Interactive systems for live events** — voting, games, audience participation. I built the interactive Harry Potter experience at WB Studio Tour Tokyo.
- **Game development** — Unity and Unreal Engine. Game jams, puzzle games, kiosk systems.
- **Web applications** — React, Node.js, Python. Subscription managers, AI social platforms, booking systems.
- **Plugins and tools** — After Effects plugins, Unity asset store tools, custom automation.

## Skills

**Languages:** C#, JavaScript, TypeScript, Python, Java, C++, Rust (learning)
